# google-reviews-download

A web tool that takes a Google Business name or Place ID and returns a downloadable CSV / JSON / XLSX export of that business's Google reviews. Powered by SemanticForce (mocked until creds land).

EMD project targeting the query "google reviews download" plus long-tail variants. Built by an autonomous Claude Code routine that runs at 04:00 Europe/Madrid each day.
